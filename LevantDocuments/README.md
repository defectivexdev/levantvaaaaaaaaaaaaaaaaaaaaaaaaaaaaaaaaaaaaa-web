# Documents

โฟลเดอร์นี้ใช้สำหรับเก็บเอกสารต่างๆ ของโปรเจค Levant Virtual Airlines

## 📁 โครงสร้าง

```
documents/
├── architecture/     # สถาปัตยกรรมระบบ
├── api/             # API documentation
├── guides/          # คู่มือการใช้งาน
├── development/     # เอกสารสำหรับนักพัฒนา
└── deployment/      # คู่มือการ deploy
```

## 📝 แนวทางการเขียนเอกสาร

- ใช้ภาษาไทยหรืออังกฤษก็ได้
- ใช้ Markdown format
- ตั้งชื่อไฟล์ให้สื่อความหมาย เช่น `database-schema.md`, `api-endpoints.md`
- เพิ่ม frontmatter ถ้าจำเป็น

## 🔗 เอกสารที่มีอยู่แล้ว (ใน root)

- `ACARS_REAL_DATA_INTEGRATION.md` - การเชื่อมต่อ ACARS
- `BLACKLIST_API_INTEGRATION.md` - Blacklist API
- `DISCORD_OAUTH_SETUP.md` - Discord OAuth
- `INNGEST_SETUP.md` - Inngest configuration
